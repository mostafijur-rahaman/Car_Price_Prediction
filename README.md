# 🚘 Car Price Predictor

### Predict the resale value of a used car using Machine Learning

A modern web application built with **Python**, **Flask**, and **Scikit-learn** that predicts the estimated market price of a used car based on its specifications.



\

---

# 📌 Overview

This project predicts the selling price of a used car using a Machine Learning model trained on historical car data.

Users enter:

* 🚗 Company
* 🚘 Car Model
* 📅 Purchase Year
* ⛽ Fuel Type
* 🛣️ Kilometers Driven

The application processes the input through a trained **Scikit-learn Pipeline** and returns the estimated market value instantly.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/mostafijur-rahaman/Car_Price_Prediction.git
```

```bash
cd Car_Price_Prediction
```

---

## Create a Virtual Environment

### Windows

```bash
python -m venv .venv
```

Activate it:

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv .venv
```

```bash
source .venv/bin/activate
```

---

## Install Required Libraries

```bash
pip install Flask pandas numpy scikit-learn
```

---

# ▶️ Run the Project

```bash
python application.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
